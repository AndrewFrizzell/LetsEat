#Let's Eat

A simple web app that helps you decide where to eat by filtering restaurants and randomly picking one for you.

##Features

- Filter restaurants by:
  - Cuisine
  - Price
  - Rating
  - Distance
- Random "Pick For Me" option
- Reset filters button
- Clean and responsive UI

##Built With

- Python
- Flask
- HTML
- CSS

##How It Works

1. Select your filters
2. Click **Filter** to view matching restaurants  
3. Or click **Pick For Me** to get a random choice
